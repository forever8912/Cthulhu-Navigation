# 《恐怖大航海》船只系统设定

## 船只基础属性

| 属性 | 说明 | 初始值范围 |
|------|------|------------|
| 生命 | 船只耐久度，归零则沉没 | 100-500 |
| 航速 | 移动速度 | 5-20 |
| 建造空间 | 可建造房间数量 | 1-10 |
| 护甲 | 减少受到的伤害 | 0-100 |

| 船员上限 | 可招募船员数量 | 5-50 |

## 船只等级系统

### 等级上限规则
- **顶级路径船**（吸血鬼、精灵、龙族、亡灵、海妖、战士、法师、牧师、盗贼）: 最高7级
- **从神级船**（狼人、巨人、矮人、航海）: 最高6级
- **生活技能船**（渔夫船、农业船、工匠船、贸易船、探索船、医疗船）: 最高5级

## 船只强化

### 属性强化
- 消耗材料永久提升船只属性
- 每项属性可强化10次
- 强化消耗随次数递增

### 附魔系统
- 消耗魔晶和神器碎片为船只附魔
- 附魔效果包括：元素伤害、生命恢复、速度提升等
- 最多可同时装备3个附魔

### 船员配置
- 不同船员提供不同加成
- 船长提供领导力加成
- 水手提供航速加成
- 炮手提供攻击加成
- 医生提供治疗加成

## 船只战斗系统

### 攻击类型
- **撞击**: 基础物理伤害
- **炮击**: 远程攻击，消耗弹药
- **技能攻击**: 特殊能力攻击

### 防御机制
- 护甲减少物理伤害
- 魔法抗性减少魔法伤害
- 闪避几率躲避攻击

### 特殊战斗状态
- **燃烧**: 持续火焰伤害
- **冰冻**: 减速效果
- **麻痹**: 无法行动
- **魅惑**: 攻击友方
